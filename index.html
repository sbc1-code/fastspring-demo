How This Integration Works

This storefront uses FastSpring's Store Builder Library (SBL) to handle product display, cart management, and checkout. The SBL loads via a single script tag and uses HTML data attributes to bind product data and actions directly to your markup, with no framework required.

<!-- 1. Load the Store Builder Library -->
<script id="fsc-api"
  src="https://sbl.onfastspring.com/sbl/1.0.3/fastspring-builder.min.js"
  data-storefront="your-store.onfastspring.com/popup-checkout"
  data-data-callback="onSessionUpdate"
  data-popup-closed="onCheckoutComplete"></script>
<!-- 2. Display product price (auto-localized) -->
<span data-fsc-item-path="neon-drift" data-fsc-item-price></span>

<!-- 3. Add to cart with one attribute -->
<button data-fsc-item-path-value="neon-drift" data-fsc-action="Add">Add to Cart</button>

<!-- 4. One-click buy (clear cart, add, open checkout) -->
<button data-fsc-action="Reset, Add, Checkout">Buy Now</button>

In production, the SBL script handles localized pricing (200+ countries, every major currency), tax calculation, and a secure popup checkout, all without redirecting the user away from your store.
